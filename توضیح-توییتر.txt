📖 راهنمای کامل: TWITTER-API-SETUP.md


خلاصه مراحل:
───────────────────────────────────────────────────────

1️⃣  ثبت‌نام در Twitter Developer
    🔗 https://developer.twitter.com
    
2️⃣  ساخت App و دریافت Bearer Token
    📝 Project Name: StyleSync
    🔑 Bearer Token را کپی کن
    
3️⃣  ساخت فایل .env
    ```
    cp .env.example .env
    ```
    
    در .env بنویس:
    ```
    TWITTER_BEARER_TOKEN=AAAAAxxxYourTokenHere...
    ```

4️⃣  تغییر کد در server.js
    
    قبل:
    ```javascript
    import { analyzeTwitterProfile }
    from './services/twitterAnalysisService.js';
    ```
    
    بعد:
    ```javascript
    import { analyzeRealTwitterProfile }
    from './services/realTwitterService.js';
    ```

5️⃣  ریستارت سرور
    ```
    pkill node
    npm run dev
    ```
