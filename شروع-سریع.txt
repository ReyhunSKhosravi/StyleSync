═══════════════════════════════════════════════════════
     ⚠️  خطاهای رایج
═══════════════════════════════════════════════════════

❌ "Cannot find module"
───────────────────────────────────────────────────────
راه‌حل:
    cd backend
    npm install


❌ "Port already in use"
───────────────────────────────────────────────────────
راه‌حل:
    lsof -i :5000
    kill -9 <PID>


❌ Quiz کار نمی‌کند
───────────────────────────────────────────────────────
راه‌حل:
    1. مطمئن شوید Backend اجراست
    2. http://localhost:5000/api/health را چک کنید
    3. Console مرورگر (F12) را ببینید


❌ صفحه سفید
───────────────────────────────────────────────────────
راه‌حل:
    1. Frontend اجراست؟ → cd frontend && npm run dev
    2. F12 → Console → ببینید خطا چیه
